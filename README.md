# Interpreters

## Prerequisites

## Linux

    sudo apt-get install opam

## Mac

    brew install opam

## All platforms

    opam init
    eval $(opam env)
    opam switch create 4.12.0
    opam install utop

Installing utop will likely take a few minutes
